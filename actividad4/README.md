# Actividad 4

## Temática
Esta actividad se vincula con la lógica difusa aplicada al desarrollo de prototipos programables.  
El material de referencia presenta dos experiencias prácticas: un prototipo para la detección del estado ambiental y un mecanismo de avance autoajustable, ambos implementados con herramientas como Matlab, SciLab y Arduino.

## Contenido
En esta carpeta se incluye el archivo PDF correspondiente a la resolución de la Actividad 4.

## Material de referencia
El artículo base aborda conceptos como:
- inferencia difusa;
- método de Mamdani;
- fuzzificación y defuzzificación;
- funciones de pertenencia;
- reglas difusas;
- implementación sobre hardware Arduino.
